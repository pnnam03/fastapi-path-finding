# fastapi-path-finding
- Clone the repo
- Python version: ``3.8.0``
- In `/fastapi-path-finding`
```
pip install -r requirements.txt
```
- Run the backend
```
uvicorn main:app --host=0.0.0.0 --port=${PORT:-5000} --forwarded-allow-ips '*'
```
